SMART: Autonomous Driving Motion Generation

Overview

Welcome to the SMART project page. SMART is a novel autonomous driving motion generation paradigm that models vectorized map and agent trajectory data into discrete sequence tokens, processed through a decoder-only transformer architecture.

Features

  • State-of-the-art performance on generative Sim Agents challenge.
  • Zero-shot generalization capabilities.
  • Scalability with over 1 billion motion tokens.

Project Details

SMART leverages a GPT-style method to train for next token prediction across spatial-temporal series, achieving remarkable inference speed and accuracy.
